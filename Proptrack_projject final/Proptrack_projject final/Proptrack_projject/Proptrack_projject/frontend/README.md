# 🏠 PropTrack - MERN Real Estate Marketplace

PropTrack is a full-stack real estate platform built using the **MERN** (MongoDB, Express, React, Node.js) stack. It allows users to browse property listings, search for specific homes, and communicate directly with an administrator through an integrated chat system.

## 🌟 Key Features

* **Dynamic Property Gallery**: Displays property cards featuring high-quality images, titles, and pricing (e.g., Rs. 400.00 Lac).
* **Smart Search & Filtering**: Includes a search-by-title bar and a location-based dropdown menu to refine property results.
* **Property Submission**: Sellers can easily list new properties using the "Add Property" management feature.
* **Real-time P2P Support**: A built-in floating chat interface ("P2P Chat Support") for direct messaging between users and Admin.
* **Optimized Chat UI**: The chat window features a responsive design with a scrollable message area and a locked input bar to ensure usability at all times.

## 🛠️ Technology Stack

* **Frontend**: React.js (utilizing Hooks like `useState`, `useEffect`, and `useRef`).
* **Backend**: Node.js & Express.js.
* **Database**: MongoDB for storing property and chat data.
* **API Management**: Axios for seamless frontend-to-backend communication.

## ⚙️ Installation & Setup

To run this project locally, follow these steps:

### 1. Backend Setup

```bash
cd server
npm install
npm start

```

*Note: Ensure your MongoDB connection string is correctly configured in your environment variables.*

### 2. Frontend Setup

```bash
cd client
npm install
npm start

```

The application will be accessible at [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000).



* **Marketplace Interface**: View the main search and property listing grid.
* **Support Chat**: View the functional P2P chat system in action.
